Pengertian GIT:
Git adalah version control system yang digunakan para developer untuk mengembangkan software secara bersama-bersama.

Mengapa dibutuhkan Git?
Karena dalam sebuah website, dibutuhkan sebuah tools yang dapat memudahkan para developer untuk memprogram website dengan cepat dan saling terintegrasi. GIT hadir sebagai salah satu version control system. Sehingga, Git ini dapat memudahkan programmer untuk mengetahui perubahan source codenya daripada harus membuat file baru.
